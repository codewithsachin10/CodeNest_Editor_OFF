# CodeNest

> Your first coding setup. Done right.

A beginner-friendly desktop app that sets up programming languages and a ready code editor in one click.

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or bun

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
